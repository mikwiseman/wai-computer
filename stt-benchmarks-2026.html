The one-line read

For top real-time English you still need a closed API (Cartesia / ElevenLabs, ~3.6%) — open models trail by ~1.6pp. For batch, open Voxtral Small (2.77%) is competitive and beats what we pay Deepgram for now. For Russian, open wins outright. So self-host for Russian + batch + control; keep a managed API only for best-in-class English real-time.
