02 · THE RUSSIAN PAIR

GigaAM vs T-one — they don't compete, they split the job

Both are open, Russian-only, and best-in-class — but at different things. Use both: GigaAM for accuracy, T-one for live.

GigaAM v3 Sber · MIT · 240M

Russian — accuracy
  • Recordings, files, async dictation where the transcript quality matters most (2.6% WER — beats every API on Russian).
  • Noisy / far-field / call-center Russian — v3 added ~30% improvement on these hard domains.
  • Small (240M) → runs on a modest GPU; RNN-T or CTC decoders, with e2e variants that add punctuation.
  • Not for sub-300ms live streaming — the card doesn't claim native streaming → use T-one.
  • Russian-only — no English / other languages.
Note: GigaAM is Sber's open ASR model — not GigaChat (Sber's LLM). Different product, same lab.

T-one T-Bank · Apache-2.0 · 71M

Russian — live / streaming
  • Real-time Russian: live captioning, voice agents, streaming dictation — native CTC streaming in 300ms chunks.
  • Telephony / call-center / phone audio — purpose-built and tuned for it (best open WER on call data).
  • Tiny (71M) → many concurrent streams per GPU, low latency, cheap to serve.
  • Not the top pick for clean read-speech accuracy — GigaAM wins there.
  • Russian-only — no English / other languages.
Trained from scratch (modified Conformer, NeMo) — not a Whisper fine-tune.
03 · THE OTHERS

Where to use everything else

VibeVoice-ASR Microsoft · MIT · 9B

Long-form + diarization
  • Long meetings / interviews needing speakers + timestamps — 60 min in one pass, incl. Russian. The best open fit for a recordings product.
  • Not real-time; 9B → needs a real GPU (not edge).

Parakeet-tdt-0.6b-v3 NVIDIA · CC-BY · 600M

English / multilingual batch — speed
  • High-volume English/EU batch — fastest open model (~30–40× Whisper), $0 marginal. Handles 25 langs incl. Russian as a fallback.
  • Not the primary Russian pick (trails GigaAM); not low-latency streaming.

whisper-large-v3-turbo OpenAI · MIT · 809M

English dictation + broad fallback
  • English push-to-talk (with Silero VAD), and any of 99 languages where you have no specialist — the safe, mature default.
  • Russian accuracy → use GigaAM; no true streaming (chunked only).

Qwen3-ASR 1.7B Alibaba · Apache · 1.7B

Broad multilingual, one model
  • EN + RU + 50 more in one model, with code-switching and streaming — when you don't want to run a per-language stack.
  • Beaten on pure Russian (GigaAM) and pure English accuracy (Granite).

Granite Speech 4.1 2B IBM · Apache · 2B

English — top accuracy
  • Most accurate open English (Open ASR Leaderboard #1, 5.33%) for quality-first English batch.
  • No Russian (6 langs); offline only, no streaming.

Managed APIs Groq · Cartesia · ElevenLabs · Deepgram

When not to self-host
  • Groq whisper-turbo ($0.04/hr) — cheap burst/overflow + fallback.
  • Cartesia Ink-2 / ElevenLabs Scribe v2 RT — best real-time English (open trails ~1.6pp).
  • Skip when you need data residency, cost-at-scale, or Russian SOTA — self-host instead.
04 · FOR WAICOMPUTER

The routing we'd ship

Language + mode router
Russian recording/file → GigaAM v3
Russian live / dictation → T-one
Russian long meeting → VibeVoice-ASR (speakers + timestamps)
English recording/batch → Voxtral Small / Parakeet-v3
English dictation → whisper-turbo + Silero VAD
Best English real-time → managed API (Cartesia / ElevenLabs)
Overflow / spikes → Groq whisper-turbo

Net: self-host for Russian (open is SOTA), recordings/batch (open beats Deepgram), and long-form+diarization; lean on a managed API only for best-in-class English real-time. One cheap 24–32 GB GPU (Hetzner GEX44 €184/mo, or RTX PRO 4500 ~$0.24/hr) covers the self-hosted models for ~1,000–3,000 daily users.
